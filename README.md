# 🔄 Git-Sync

**Git-Sync** is a modular Python CLI tool designed to simplify and automate repetitive Git, system, and DevOps-related tasks. Built using `Typer`, `Rich`, and `GitPython`, this tool helps developers manage Git workflows, run system diagnostics, and test APIs—all from a single unified CLI.

## 🔧 Features

- 🚀 Git Automation (clone, commit, push, sync)
- 🛡️ System Health Check (CPU, memory, disk stats)
- 🌐 API Tester with custom endpoints
- 📦 Modular structure for easy extensions
